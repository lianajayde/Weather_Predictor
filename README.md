# Weather_Predictor

# Details
Weather Predictor is an application created to allow you to view a city's weather forecast currently and for five days in the future.

# Technologies Used
-JavaScript<br>
-HTML and CSS<br>
-Weather API


Helpful websites used..<br>
-Bootstrap<br>
-W3Schools<br>
-jQuery<br>
